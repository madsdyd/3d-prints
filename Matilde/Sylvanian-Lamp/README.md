# Lamp for Sylvanian dolls

This is a lamp designed and printed for my daughters Sylvanian dolls. The lamp is in two parts, and connects by screwing together. You can print the parts in various colors and combine them.

Attribution: This uses the [Thread library](http://www.thingiverse.com/thing:8793). Its included in the source.

# Instructions

0.3mm layer height 15% PLA was fine for me.
